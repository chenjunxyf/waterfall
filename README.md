# 图片等高不等宽瀑布流

## 功能

* 实现等高图片的瀑布流

## 目录说明

```
- mock -------------------- 模拟数据
    + html ---------------- 同步模拟数据
    + ajax ---------------- 异步模拟数据
+ page -------------------- 页面模板
+ static ------------------ 静态资源
- fat.config.js ----------- fat工具配置文件
- fis-conf.js ------------- fis3b配置文件
- fis-remote-conf.js ------ fis3b开发机配置
```

## 使用说明

* 安装[fat-byte](https://github.com/chenjunxyf/fat)
